### 知识点
1. less使用for循环

```css
     /* less定义循环 */
     div{
        .loop(@n) when (@n <=4) {
            /**注意 ：(@{n}) 形式 */
            &:nth-child(@{n}) {
                animation: here-am-i .5s ease-out (@n / 5s);
                animation-fill-mode: both;
            }

            .loop((@n + 1));
        }

        .loop(2);
     }
```
